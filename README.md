# Tableau Airbnb Dashboard

This repository contains a Tableau dashboard project based on the Airbnb Listings dataset.

Download
You can download the Tableau workbook from Google Drive:
https://drive.google.com/file/d/1OCxwIr2ywlpndJ0Qw7tkx66qBwzfDyk2/view?usp=sharing 

Project Summary
The project explores how room type, neighbourhood, location, and bedroom count influence Airbnb pricing patterns.

Notes
This file can be opened in Tableau Desktop.
